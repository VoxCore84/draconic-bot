# DraconicBot Identity and Commands
Tags: bot, draconicbot, help, commands
Last reviewed: 2026-03-12
Owner: DraconicWoW Staff

## Facts
- DraconicBot is the official support bot for the DraconicWoW private server community.
- It provides automated help with server setup, troubleshooting, GM commands, and general WoW private server questions.
- It uses AI-powered responses to diagnose issues and guide users through common problems.
- DraconicBot monitors support channels and proactively answers questions when it detects a support-related message.
- The bot can also be mentioned directly (@DraconicBot) or replied to for follow-up questions.

## Slash Commands
- `/help` -- Shows available commands and how to get support.
- `/troubleshoot` -- Starts an interactive troubleshooting session for common issues.
- `/spell [name or id]` -- Looks up spell information from the game database.
- `/item [name or id]` -- Looks up item information from the game database.
- `/creature [name or id]` -- Looks up creature/NPC information from the game database.
- `/area [name or id]` -- Looks up zone or area information.
- `/faction [name or id]` -- Looks up faction information.
- `/gmcommand [command]` -- Shows usage and details for a specific GM dot-command.
- `/buildcheck` -- Checks the current server build number and compares it to the expected client build.
- `/about` -- Shows bot version, uptime, and project information.
- `/announce` -- (Staff only) Sends an announcement to a channel.
- `/faqstats` -- Shows statistics on the most common questions and AI usage.

## DM Setup Wizard
- New users can DM the bot with `!setup` to start a guided setup wizard.
- The wizard walks through client installation, Arctium setup, server connection, and account creation step by step.
- Each step includes verification checks so the user can confirm before moving on.

## Escalate when
- The user asks about a command that is not listed here.
- The user needs help with something the bot cannot automate (e.g., account recovery).
